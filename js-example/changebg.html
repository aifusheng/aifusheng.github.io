通过更换类名来更改背景色

有的网页根据白天还是黑夜可以选择切换背景,下面来看看它的实现原理.
实现原理:
当点击切换背景按钮时,触发onclick事件,给onclick事件,绑定修改样式类名的函数,从而实现切换背景.

查看演示 下载源码

下面是js源代码：

var x=0;
var bg=new Array();
bg[0]="bg1"
bg[1]="bg2"
bg[2]="bg3"
function modifyclass(){
	x+=1
	if (x==3){
		x=0;
	}
	document.getElementById("modify").className = bg[x];

}
